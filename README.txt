This repository hosts simple scripts to backup the NTC CHIP related software.
To use them you will need:

  * Python
  * Bash
  * wget
  * GIT

You can check the individual scripts for their requirements.

These scripts are being provided free of charge with absolutley no warranty.
Use at your own risk.

Manifest:

  README.txt: This file.
  down_chrome_images.py: backup images on flash.getchip.com
  down_cli_images.sh:    backup images used by chip-update-firmware.sh
  ntc_gits.sh:           does a git clone --mirror for listed NTC github repos.
